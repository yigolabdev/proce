# Promptor Frontend (Proce)

React 19 + Vite + TypeScript + Tailwind v4 starter focused on planning-first UI.

## Getting Started

```bash
npm install
npm run dev
```

## Tech Stack
- React 19, Vite 7, TypeScript 5.9
- Tailwind CSS v4
- React Router v7
- TanStack Query v5
- ESLint (flat) + Prettier (+ Tailwind plugin)

## Conventions
- Design system: Promptor — Primary `#3D3EFF`, rounded-2xl, gradients, soft animation
- Use `src/components/ui` for primitives (e.g., Button)
- Providers live in `src/providers`

## Scripts
- `npm run dev` — start dev server
- `npm run build` — typecheck and build
- `npm run preview` — preview build
- `npm run lint` — run ESLint
