## Executive Summary

**Proce**는 AI 기반 전략적 성장 플랫폼으로, 기업의 비전을 실행 가능한 목표로 전환하고, 실행 데이터로부터 학습하여 지속적으로 성과를 개선하는 SaaS 솔루션입니다.

### 핵심 수치

- **TAM (Total Addressable Market)**: $15B (글로벌 업무 관리 소프트웨어 시장)
- **SAM (Serviceable Available Market)**: $4.5B (AI 기반 전략 관리)
- **SOM (Serviceable Obtainable Market)**: $450M (5년 목표)
- **타겟 가격**: $15-35/사용자/월
- **목표 LTV/CAC**: 3.0+
- **목표 Net Revenue Retention**: 120%+

### 비즈니스 모델

- **수익 모델**: Subscription SaaS + AI Usage-Based Pricing
- **가격 전략**: Freemium → Paid Plans (4단계)
- **시장 접근**: Bottom-up (팀) + Top-down (엔터프라이즈)

### 시장 기회

#### 1. 미충족 수요 (Unmet Needs)

1. **전략-실행 단절**
   - 현황: 67%의 기업이 전략과 실행의 연결 실패
   - 기회: 전략 정렬 자동화

2. **수동 분석 부담**
   - 현황: PM/PO가 주당 8-12시간 수동 분석
   - 기회: AI 자동 인사이트

3. **예측 불가능성**
   - 현황: 58%의 프로젝트가 예산/일정 초과
   - 기회: AI 예측 및 시뮬레이션

4. **학습 사이클 부재**
   - 현황: 성공/실패 패턴 재현 불가
   - 기회: 자동 패턴 학습

#### 2. 시장 트렌드

1. **AI-First 시대**
   - 84%의 경영진이 AI를 경쟁력으로 인식
   - 생성형 AI 통합 솔루션 수요 급증

2. **데이터 기반 경영**
   - Real-time 데이터 기반 의사결정
   - 예측적 관리 수요

3. **원격/하이브리드 업무**
   - 비동기 협업 도구 필요
   - 투명성 및 가시성 중요

4. **Vertical SaaS 성장**
   - 범용보다 특화 솔루션 선호
   - 산업별 맞춤형 수요

## 가치 제안

### Proce의 고유 가치 (Unique Value Proposition)

> **"AI가 과거로부터 학습하여, 미래 성공을 예측하는 유일한 전략 실행 플랫폼"**

#### 핵심 가치 3가지

1. **완벽한 정렬 (Perfect Alignment)**
   - 비전 → 전략 → KPI → OKR → Task의 끊김없는 연결
   - 실시간 정렬도 모니터링
   - 미연결 목표 자동 탐지

2. **예측적 관리 (Predictive Management)**
   - AI가 목표 달성 가능성을 사전 예측
   - 리스크 조기 경고 (2주 이상 앞서)
   - What-if 시뮬레이션

3. **자동 학습 (Auto Learning)**
   - 성공/실패 패턴 자동 학습
   - 주간/월간/분기 인사이트 자동 생성
   - 다음 분기 목표 자동 추천

### 고객 Pain Points vs Proce 솔루션

| 고객 문제 | 현재 솔루션의 한계 | Proce 솔루션 | 결과 |
|----------|-----------------|-------------|------|
| 전략과 실행 단절 | 수동 정렬, 불투명 | AI 자동 정렬 체크 | 정렬도 90%+ |
| 목표 설정 비현실성 | 감으로 설정 | AI 달성 가능성 예측 | 달성률 30%↑ |
| 분기 말 리스크 인지 | 사후 대응 | 실시간 예측 경고 | 조기 대응 |
| 수동 분석 부담 | PM이 수동 분석 | AI 자동 인사이트 | 시간 90%↓ |
| 같은 실수 반복 | 학습 사이클 부재 | 자동 패턴 학습 | 지속 개선 |

## 가격 전략

### 가격 철학

1. **가치 기반 가격** (Value-Based Pricing)
   - 고객 ROI: 10x+
   - 절감 비용 대비 20-30% 가격 책정

2. **단계적 성장** (Land & Expand)
   - 낮은 진입 장벽 (Free/Starter)
   - 사용량 증가에 따른 자연스러운 업그레이드

3. **투명한 가격** (Transparent Pricing)
   - 숨겨진 비용 없음
   - 명확한 플랜별 기능 차이

### 가격 플랜

#### 1. Free Plan (Forever Free)

**가격**: $0

**포함 내용**:
- 최대 5명
- 프로젝트 5개
- OKR 10개
- 기본 대시보드
- 모바일 앱
- AI 크레딧: 월 50 크레딧

**제한**:
- 고급 AI 기능 없음
- 데이터 보관: 3개월
- 지원: 커뮤니티

**타겟**: 프리랜서, 소규모 팀, 체험 사용자

**전환율 목표**: 15% → Starter

### 세부 계산

#### CAC (Customer Acquisition Cost)

**Year 1**:
```
마케팅 비용: $200K
영업 비용: $150K
총 S&M: $350K
신규 고객: 700개
CAC = $350K / 700 = $500
```

**Year 3**:
```
마케팅 비용: $1.2M
영업 비용: $800K
총 S&M: $2.0M
신규 고객: 5,000개
CAC = $2.0M / 5,000 = $400
```

#### LTV (Lifetime Value)

**가정**:
- 평균 ARPU (Average Revenue Per User): $25/month
- 평균 팀 크기: 30명
- 평균 MRR/고객: $750
- Gross Margin: 80%
- 평균 고객 수명: 36개월 (Year 1), 60개월 (Year 3)

**Year 1 LTV**:
```
LTV = ARPU × 고객수명 × Gross Margin
LTV = $750 × 36개월 × 0.75 = $20,250 (회사)
LTV/user = $20,250 / 30명 = $675/user
사용자 기준 LTV = $25 × 36 × 0.75 = $675/user (동일)
고객 기준 환산 = $675 × 30 = $20,250

LTV/CAC = $20,250 / $500 = 40.5x (회사 기준)
또는 개별 사용자로 계산 시 고려
```

**재계산 (더 보수적)**:
```
평균 고객 계약: $750/month
고객 수명: 24개월
Gross Margin: 75%
LTV = $750 × 24 × 0.75 = $13,500

LTV/CAC = $13,500 / $500 = 27x (여전히 매우 높음)
```

**사용자당 기준으로 재계산**:
```
평균 ARPU: $25/user/month
사용자 수명: 30개월
Gross Margin: 75%
LTV/user = $25 × 30 × 0.75 = $562.5

팀 평균 크기: 30명
LTV/회사 = $562.5 × 30 = $16,875

LTV/CAC (회사) = $16,875 / $500 = 33.8x
```

**최종 조정** (더 현실적):
```
월 ARPU: $22/user (할인 등 고려)
Gross Margin: 75%
고객 수명: 24개월
연간 Churn: 25%

LTV = $22 × 24 × 0.75 = $396/user
팀 크기: 30명
LTV/회사 = $396 × 30 = $11,880

LTV/CAC = $11,880 / $500 = 23.8x (회사 기준)
```

**더 보수적 재계산**:
```
만약 CAC를 팀(회사) 기준이 아닌 사용자당으로 환산:
CAC/user = $500 / 30 = $16.67/user
LTV/user = $396
LTV/CAC = $396 / $16.67 = 23.8x (동일)

하지만 일반적으로 SaaS에서:
- B2B의 경우 회사(계정) 기준으로 계산
- $500 CAC는 회사 전체 획득 비용
- $11,880 LTV는 해당 회사 평생 가치
따라서: LTV/CAC = 23.8x

그러나 이는 비현실적으로 높으므로 재검토 필요
```

이것도 높으므로 수정:
